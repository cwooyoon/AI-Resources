## 파이썬 용어들

### 표현식(Expression)
* 어떤 값을 만들어내는 간단한 코드
* 값의 예
```
273
10 + 20 + 30 = 10
"Python Programming"
```
* 표현식 -> 문장 -> 프로그램

### 키워드(Keyword)
* 특별한 의미가 부여된 단어로, 파이썬이 만들어질 때 이미 사용하겠다고 예약해 놓는 것
  - https://wikidocs.net/20557
```
import keyword
# print(keyword.kwlist)
kwlist = keyword.kwlist
for i in range(0,len(kwlist)):
    print("[{:10}]".format(kwlist[i]), end=" ")
    if (i+1)%5==0: print()
```
